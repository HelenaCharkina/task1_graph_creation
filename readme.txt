Создание графа

Правило: Пусть во входе была описана дуга, в которой
{ from:"A", to:"B", index:N }
то на выходе, у элемента массива, у которого поле name равно "B", в массиве
поля inputFrom под индексом N (применяется нуль индексация) должно
стоять такое число, что у выходного массива элемент с данным индексом
имет поле name, равное "A".
